About SecureCrypt

Secure, client-side AES-GCM encryption tool

What is SecureCrypt?

SecureCrypt is a web-based encryption tool that provides military-grade AES-GCM (Advanced Encryption Standard - Galois/Counter Mode) encryption for your text data. All encryption and decryption operations happen securely on the server using industry-standard cryptographic libraries.

AES-GCM Encryption

Uses 256-bit AES-GCM for authenticated encryption with integrity checking.

Secure Key Generation

Generates cryptographically secure random keys using your system's CSPRNG.

Password-Based Keys

Derive encryption keys from passwords using PBKDF2 with 310,000 iterations.

Open Source

Built with Flask and Python's cryptography library. Fully transparent code.

How It Works

1

Key Generation

Generate a secure 32-byte key or derive one from a password using PBKDF2.

2

Encryption

Your text is encrypted using AES-GCM with a unique nonce for each operation.

3

Secure Output

Encrypted data is base64-encoded for easy sharing and storage.

4

Decryption

Only with the correct key can the original text be recovered.

Security Notes

Important: This tool is for educational and personal use.

  • Always save your encryption keys securely
  • Never share encrypted data with the key
  • Use strong, randomly generated keys
  • Consider additional security measures for sensitive data

Technical Details

Backend Stack:

  • Framework: Flask (Python)
  • Crypto Library: cryptography (AES-GCM, PBKDF2)
  • Key Size: 256 bits (32 bytes)
  • Nonce Size: 96 bits (12 bytes)

Features:

  • Authenticated encryption (confidentiality + integrity)
  • Base64 URL-safe encoding
  • Proper error handling
  • Responsive web interface
